# ZFS_K
ZFS_K is a superset of ZFS adding keybord shortcuts. It's also written in C so it also fast.

# WARNING
ZFS_K is only for windows as it uses 'windows.h'. It's only dependency is the <a href="https://github.com/985479861-TXT/ZFS-CLI">ZFS<a> cli, so make sure to install it.

# Setup
Run it by running the zfs_k.exe file, nothing will happen but it will continue to run in the background. 

# Usage 
After running it press Alt + S on the keyboard to open the terminal, type any zfs arguments without the 'zfs' (like "-w yt" instead of "zfs -w yt") and it shall work.

# Side note
If for any reason you want to stop it from running in the background, just run the KRONOS.bat to stop it.
